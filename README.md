#### Ctrl+P              -> Files
#### Ctrl+T              -> Symbols/Functions
#### Ctrl+Shift+P        -> Command palette
#### Ctrl+Tab            -> buffers

#### Ctrl+Shit+M         -> diagnostics panel (new tab)
#### F8 / Shift+F8       -> Navigate diganostics        -> ESC salir de ese modo de navegacion

#### Ctrl+Shift+F        -> Es una busqueda general pero por dentro se hacer con ripgrep
                   
#### Ctrl+Shift+E        -> cambiado foco entre navegador de archivos / editor 
####                     -> En el navegador de archivos :
####                        -> Enter -> Open 
####                        -> Space -> Preview
####                        -> F2    -> Rename
                   
#### Ctrl+k + Arrow      -> Dividir panel
#### Ctrl+k + Ctrl+Arrow -> Navegar por paneles

#### Ctrl+I              -> Signature help
#### Ctrl+space          -> LSP Code completion

#### Ctrl+Shift+g        -> git panel -> Enter en un file -> diff view
